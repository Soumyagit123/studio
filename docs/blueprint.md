## Style Guidelines:

- Primary color: Deep indigo (#4B0082) to convey professionalism and intelligence.
- Background color: Dark slate gray (#2F4F4F) for a modern, sophisticated dark theme. Light theme will use a very light gray.
- Accent color: Electric purple (#BF00FF) to highlight key elements and CTAs with a vibrant contrast. 
- Body and headline font: 'Inter', a grotesque sans-serif, is suitable for both headlines and body text, maintaining a modern and objective feel throughout the website.
- Use FontAwesome icons for skills, certifications, and social links, ensuring consistency and scalability.
- Implement a responsive, grid-based layout to ensure optimal viewing experience on all devices. Utilize a sticky navbar for easy navigation.
- Incorporate subtle scroll animations to enhance user engagement and create a dynamic browsing experience.
